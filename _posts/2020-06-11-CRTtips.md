# CRT 快捷键
## 切换标签页
```
Alt + 1 / 2 /3 /4/ 5
```
切换到 1 /2/ 3/ 4/5标签页
## 建立链接
```
Alt + C
```
在左侧窗口中建立新连接

# SourceInsight 快捷键
```
ALT + , JUNP BACKWARD

ALT + . JUMP FORWARD

CTRL + = JUMP TO DEFINITION

F5 OR CTRL + g JUMP TO THE LINE 
```
